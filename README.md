# sturdy-octo-waddle

A full-stack forum application built with Next.js, Convex, and Clerk Authentication.

## Features
- **Next.js App Router** for modern routing and server-side rendering
- **Convex Database** for real-time data and file storage
- **Clerk Auth** for Google and Email login
- **Tailwind CSS** for rich aesthetics

## Getting Started

1. Initialize Convex: `npx convex dev`
2. Start the dev server: `npm run dev`
3. Open `http://localhost:3000`
